# Sistem Booking Barbershop / Salon

Aplikasi Python berbasis CLI untuk melakukan booking layanan barbershop/salon.

## Fitur
- Manajemen pelanggan
- Lihat layanan
- Booking layanan
- OOP & Modular Programming

## Cara Menjalankan
1. Import database MySQL
2. Install dependency: pip install -r requirements.txt
3. Jalankan: python main.py
